# TattleTotem

![TattleTotem](tt.png)

A "tattle-tale" addon for Turtle WoW that announces raid member mistakes in Naxxramas via /yell, /say, or /rw.



## Usage

### Slash Commands

- `/tt` or `/tattletotem` - Open configuration window
- `/tt enable` - Enable the addon
- `/tt disable` - Disable the addon
- `/tt debug` - Toggle debug mode (monitors everywhere, outputs to local chat)
- `/tt status` - Show current status
- `/tt test` - Send a test message

### Minimap Button

- **Left-click**: Open configuration
- **Right-click**: Toggle addon on/off
- **Drag**: Move button around minimap
